# Other Expert Systems

This directory contains separate expert systems that are not part of the main GeneLingua language learning application.

## Files

- `experts_voice.py` - Real-time voice interaction system
- `healthcare_expert.py` - Healthcare conversation expert
- `it_backend_interviewer.py` - IT backend interview expert

## Running

To run these expert systems, you'll need additional dependencies:

```bash
pip install pyaudio speech-recognition
```

Then run from this directory:

```bash
cd other_experts
python experts_voice.py
```

## Note

These are separate from the main GeneLingua app and have their own dependencies and functionality.
